# World Cup Dashboard
## Monitor the world cup from a simple dashboard

![Screenshot Demo](photos/image.png)

Are you to busy to sit down and watch a whole 90 minute world cup match? Would you rather be coding? Then look no further, this World Cup dashboard displays all of the games and their updates as they happen. And the dashboard can easily be moved to a second monitor to allow you to focus on whatever you're working on.

## Installation

| Windows | Mac | Linux |
| :---: | :---: | :---: |
| [Release here](https://github.com/jdszekeres/world_cup_dashboard/releases) | Build from source | build from source |

### Building from source

0. [Install Python](https://www.python.org/downloads/)
1. Clone the repository
2. Navigate to the directory: `cd world_cup_dashboard`
2. Install requirements; run `pip install -r requirements.txt` in the terminal
3. Run with the command `python main.py`
